# shell
分享自用的shell脚本

安装下载工具

apt update -y  && apt install -y curl

curl -sS -O https://raw.githubusercontent.com/rockmetalcn/shell/main/rockmetal.sh && chmod +x rockmetal.sh && ./rockmetal.sh
